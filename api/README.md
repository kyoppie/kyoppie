# kyoppie-api

[![Build Status](https://travis-ci.org/kyoppie/kyoppie-api.svg?branch=master)](https://travis-ci.org/kyoppie/kyoppie-api)
  
kyoppie is a Yuru Open Source SNS.

## how to run
前提:コンピュータにMongoDBとRedisがインストールされていて、実行されていること

```
npm install
npm run migrate
npm start
```
